Formatações principais yare

Negrito

Maneira semântica: <strong>Olá </strong>
Maneira não semântica: <b>Oláa</b>

Itálico (ênfase)

Maneira semântica: <em>Oioi</em>
Maneira não semântica: <i>Oioi</i>

Marcador de texto

Um texto marcado é usado para dar mais destaque a ele. Para isso, usamos a tag <mark>
Exemplo: Olá, este é um texto marcado.
Outro exemplo: Dá pra mudar a cor da marcação também, mas com CSS.

Textos grandes e pequenos

Também é possível fazer textos grandes e textos pequenos com as tags <big> e <small>, respectivamente.

Texto deletado

É um texto que tá lá, mas que deve ser desconsiderado. Usa-se <del>

Texto inserido

É um texto sublinhado de forma semântica. Para isso, usa a tag <ins>
Existe a tag <u>, que também é usado para sublinhar textos, mas essa tag não é semântica.;

Texto sobrescrito

É usado em textos como x6-3
Usa-se a tag <sup>

Texto subscrito

É usado em textos como H20
Usa-se a tag <sub>

IMAGENS E FAVICONS

Aqui vamos aprender como inserir imagens :>
(e também favicons)

Para favicons é: <link:favicon (<link rel="shortcut icon" href="iconehtml.ico" type="image/x-icon">)>

Para imagens é: <img scr="CTRL+ESPAÇO pra selecionar a imagem" alt="texto alternativo">

Dá também pra escolher imagens externas com links.
